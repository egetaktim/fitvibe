# Manuel GitHub Yükleme (En Kolay Yol)

Push işlemi çalışmıyorsa, dosyaları GitHub web arayüzünden manuel olarak yükleyebilirsiniz:

## Adım 1: GitHub Repository'ye Gidin
https://github.com/egetaktim/fitvibe

## Adım 2: Dosya Yükleme
1. "Add file" butonuna tıklayın
2. "Upload files" seçeneğini seçin
3. Şu dosyaları sürükleyip bırakın:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `netlify.toml`
   - `README.md`

## Adım 3: Commit
1. Aşağıda "Commit changes" bölümüne gidin
2. Commit message: "Update FitVibe app - Restore iPhone frame"
3. "Commit changes" butonuna tıklayın

Bu şekilde dosyalar GitHub'a yüklenecek ve Netlify otomatik deploy edecek!
