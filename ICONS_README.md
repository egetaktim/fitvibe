# PWA Icon Oluşturma

PWA için icon dosyaları gerekiyor. İki seçenek:

## Seçenek 1: Online Icon Generator (Önerilen)

1. https://realfavicongenerator.net/ adresine gidin
2. FitVibe logosu veya istediğiniz bir görsel yükleyin
3. Icon'ları oluşturun ve indirin
4. `icon-192.png` ve `icon-512.png` dosyalarını proje klasörüne koyun

## Seçenek 2: Basit Icon Oluşturma

Basit bir icon için:
- 192x192 ve 512x512 piksel boyutunda PNG dosyaları
- Arka plan: #667eea (mor) veya #1a1a1a (koyu gri)
- İçerik: FitVibe logosu veya "FV" harfleri

## Geçici Çözüm

Icon dosyaları olmadan da PWA çalışır, sadece "Add to Home Screen" özelliği tam çalışmaz.
